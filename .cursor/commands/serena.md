Prefer Serena MCP edit tools for writes when they fit the requested change: replace_regex for targeted text/block replacements, replace_symbol_body for whole-symbol rewrites, insert_before_symbol/insert_after_symbol for symbol-relative insertions, and create_text_file for complete new text files.  lean-ctx remains the preferred read/search/shell/tree layer.
